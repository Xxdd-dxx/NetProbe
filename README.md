# TCP/UDP 协议网络调试助手兼数据转换和校验

这是一个基于 C++ 和 Qt5 编写的轻量级、模块化的网络调试与数据处理工具。主要用于在嵌入式开发、物联网设备调试或网络编程学习中，快速进行 TCP/UDP 报文的收发测试、进制转换以及数据校验。

## 🌟 核心功能

1. **TCP 服务器 (TCP Server)**
   * 支持多客户端并发连接。
   * 支持自定义监听 IP 和端口。
   * 支持向所有已连接客户端广播发送数据。
2. **TCP 客户端 (TCP Client)**
   * 支持连接指定 IP 和端口的 TCP 服务端。
   * 内置 3 秒连接超时保护机制，防止界面假死。
   * 完善的异常断线处理与日志提示。
3. **UDP 服务器 (UDP Server)**
   * 绑定本地端口进行监听。
   * 自动记录最后一次发送数据的客户端 IP 和端口，方便快捷回复。
4. **UDP 客户端 (UDP Client)**
   * 向指定目标 IP 和端口发送无连接的数据报文。
5. **数据转换与校验工具 (Data Tool)**
   * 支持进制互相转换：十进制转二进制、二进制转十进制。
   * 支持字符串与十六进制互相转换（自动过滤非法空格与换行符）。
   * 集成常用校验和与哈希算法：CRC16 (Modbus)、CRC32 (IEEE 802.3)、MD5、SHA1、SHA256。
   * 支持校验结果一键大写显示与一键复制。

## 🛠️ 技术栈

* **语言:** C++
* **框架:** Qt 5.x (Core, Gui, Network, Widgets)
* **架构:** 采用模块化设计（Component-based），将各个网络模块与 UI 解耦，代码结构清晰，适合新手学习参考。

🚀 编译运行步骤

1.安装 Qt Creator 并配置好编译器（如 MinGW 7.3.0 或 MSVC）。
2.克隆本项目到本地。
3.使用 Qt Creator 打开 NetworkAssistant.pro。
4.点击左下角的 构建 (Build) 按钮。
5.点击 运行 (Run) 即可启动助手。

📖 开发者心得

1.纯代码布局: 这个项目没有用 .ui 文件，虽然写起来代码量变多了，但对 Layout（布局管理器）的理解深了很多，界面拉伸也不会乱了。
2.异步超时处理: 以前连接服务器失败会卡死好久，现在加了 QTimer 做异步超时，体验好多了。
3.从“能跑”到“好用”：最初的十六进制转换经常因为我随手打的一个空格就报错。后来我学到了用正则表达式或者 replace 函数先做数据清洗，这让我意识到用户输入永远是不可信的，代码必须具备鲁棒性。
4.细节优化: 发现很多人复制 Hex 数据时会带空格，导致转换报错。我在底层加了正则和字符替换逻辑，现在带空格也能正常转换了。
